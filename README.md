## Installation

- Front-End

  navigate to frontEnd folder

  `cd frontEnd`

  install npm dependencies

  `npm install`

  start the front-end-app

  `npm start`

- Back-End

  navigate to back-end folder

  `cd back-end`

  install npm dependencies

  `npm install`

  start the back-end-server

  `npm start`
